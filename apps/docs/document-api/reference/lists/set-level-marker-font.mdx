## Summary

Set the font family used for the marker character at a specific list level.

- Operation ID: `lists.setLevelMarkerFont`
- API member path: `editor.doc.lists.setLevelMarkerFont(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`
